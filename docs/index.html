ASE285-ChatGPT-clone Architecture & User Manual

🏗 Design & Architecture

The application utilizes a Decoupled Full-Stack Architecture, separating the React-based frontend from the Node.js/Express backend to ensure modularity and ease of maintenance.

Simplified File Structure

my-app
├── client/          # React Frontend (UI & State)
│   ├── config/      # API Keys (gemini.js)
│   └── src/         # Components & Context
├── server/          # Node.js/Express Backend
│   ├── data/        # SQLite Storage
│   └── server.js    # API Endpoints
└── run.sh           # Main Execution Script

Technical Overview

  • Frontend: Managed via React Context API to handle global chat state. Styles are component-scoped for clarity.
  • Backend: Acts as a secure proxy to the Gemini API and manages local persistence via SQLite.
  • Integration: Seamless communication via RESTful API calls between the client and server.

📖 User Manual

Follow these steps to get the application running on your local machine.

Step 1: Get your API Key

Go to Google AI Studio. Click on "Get API key" in the sidebar. Create a new key and copy it to your clipboard.

Step 2: Configure the Client

Open my-app/client/config/gemini.js in your editor and paste your key:

export const apiKey = "YOUR_API_KEY_HERE";
Step 3: Launch the App

Open a terminal in the my-app folder and run:

.
un.sh
Prerequisites: You must have Node.js installed. Ensure you have executed npm install in both the /client and /server directories prior to running the launch script.
